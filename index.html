Vue.js != full-blown framework

Vue.js is not a complete framework, its focus is only the view layer

  • Easy to implement with other libraries or existing projects
  • When used with appropriate tools and support libraries, it is also capable of supporting sophisticated one-page applications (Single Page Application, SPA)

MVVM (Model - ViewModel - Model)

Why Vue.js and not Angular?

  • Simples
  • More Flexible
  • Clearer separations between directives and components
  • Best performance
    • Do not use Dirty Checking(Digest cycle)
    • Utilize accessors (getters and setters)

What is reactive programming?

Reactive programming

It is based on data flows and the propagation of changes, asynchronous and non-blocking

  • React to events - The nature of being event-oriented allows the other qualities
  • React to loads - Focus on scalability rather than performance targeting a single user
  • Reacting to faults - Be resilient with the ability to recover at all levels
  • React to users - Combine the above features to provide an interactive user experience
Combination of competing systems, event-oriented systems and asynchronous systems

Reactive data binding

  • Keeps your data and the DOM in sync
  • jQuery is imperative, repetitive and prone to errors
  • Special syntax for bind
  • Easier to write, understand and maintain code
